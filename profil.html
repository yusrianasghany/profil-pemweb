Yusrian Asghany

S1 Teknik Informatika di Universitas Brawijaya

Halo saya Yusrian, saya berasal dari Kota Malang, dan saat ini juga tinggal di Malang bersama keluarga. Saya lahir pada 25 Oktober 2002. Sejak SD, saya memiliki ketertarikan pada pada bidang matematika. Saya juga memiliki ketertarikan pada fisika. Cara bagaimana manusia memahami dunia melalui fisika dan matematika membuat saya takjub dan bersyukur. Profesi impian saya adalah menjadi seorang Data Engineer, kerena profesi ini sesuai dengan minat saya dan berpotensi besar memberikan finansial yang baik.

Selama 12 tahun sekolah, saya merasa cukup baik pada bidang akademik. Saya mendapatkan peringkat 3 di SD, peringkat 6 di SMP, dan peringkat 1 di SMA. Pencapaian ini membuat saya bangga dan bersyukur terhadap nikmat Tuhan yang diberikan kepada saya, selain itu membanggakan orang tua juga membuat saya bangga terhadap diri saya sendiri. Saya juga bangga bahwa saya mampu menjadi seorang ketua OSIS semasa SMP dulu. Mengemban tugas menjadi ketua OSIS banyak memberikan saya pembelajaran, terutama mengenai softskill, saya lebih memahami apa itu tanggung jawab, kepemimpinan, dan kedisiplinan. Saya bangga bahwa selama ini saya selalu berusaha berperilaku baik kepada orang lain dan menaati hukum.

Menjadi mahasiswa di Teknik Informatika Universitas Brawijaya juga merupakan kebanggaan bagi saya. Meskipun begitu, di masa awal kuliah saya sering merasa rendah diri melihat banyak teman-teman kuliah saya yang keterampilan dan pengetahuannya sangat baik. Saya juga tidak aktif berorganisasi karena menjadikan kuliah daring sebagai alasan.

Semakin kesini saya menyadari bahwa kuliah daring tidak bisa saya jadikan alasan untuk tidak berkembang di kampus. Kesempatan berkuliah tidak bisa didapatkan semua orang, dan saya tidak ingin menyia-nyiakan kesempatan yang saya miliki ini. Saya berusaha menanamkan keyakinan pada diri saya, bahwa semua hal itu mungkin jika kita mau berusaha dan tetap berdoa. Bagi saya yang sekarang, kampus tidak hanya tempat untuk berkompetisi, melainkan juga tempat untuk mengembangkan diri sesuai bakat dan minat. Saya menjalani perkuliahan saya dengan sungguh-sungguh. Pada semester ini, saya berencana untuk mengikuti beberapa organisasi di kampus. Saya ingin bisa berkontribusi di lingkungan kampus. Saya harap, usaha saya dapat mengantarkan saya mencapai impian saya.
